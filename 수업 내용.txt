* 개발 방법   
1.  jsp / 서블릿   :  서버 설정(톰캣). 서블릿은 db 연동, 데이터는 jsp로 전달 ->html 화면
   (스프링부트)   :  GET, POST

                         html    <-->   jsp  <-- (dto)  --> 서블릿  <--(dto)--> db

1)  jsp(html의 a 태그, input ) 에서 서블릿으로 Query 문자열 🔥 파라미터 🔥 가져오기
    ㄴ 서블릿은 파라미터값을 검색에 사용하기 (form 사용 시 모두 가능)
2) 서블릿에서 jsp 로 attribute 데이터 전달하기. 
    ㄴ jsp 에서는 같은 이름의 🔥 애트리뷰트 🔥 가져와서 태그 요소로 출력


2. 리액트/서블릿 : 서블릿은 db연동하여 얻은 데이터를 json 문자열로 응답(REST API)
                         리액트를 웹서버 프론트엔드에서 api 요청 응답으로 화면 구현
   리액트/스프링부트
   => REST API  요청 방식은 GET,POST,PUT,PATCH,DELETE ...
